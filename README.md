# Exercício 3 - Guilherme Ditzel

Para rodar o programa você vai precisar instalar Node.js 

    https://nodejs.org/en/
 
Em seguida, após instalado utilize dois terminais diferentes, poruqe em um estará rodando o backend e no outro o frontend

No primeiro, execute os seguintes comandos:

    cd Back
    npm install
    npm run start
 
No segundo, execute os seguintes comandos:
 
    cd Front
    npm install
    npm run start
    
Após o frontend rodar ele deverá abrir uma pagina no navegador no endereço "localhost:4200/log-in",
caso não abra automaticamente, acesse esse endereço.

Lá deverá se encontrar a página q acessará o backend
